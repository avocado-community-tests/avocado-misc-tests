description:
------------------------
This Program to check network configuration details like  speed, driver name, businfo, hardware address, transmission mode, link state using ethtool and ifconfig.
-----------------------------
Inputs Needed To Run Tests:
-----------------------------
interface --> host interface to perform test
Host-IP   --> Specify the IP for ip configuration of host.
Netmask   --> Specify the Netmask for ip Configuration of host.

-----------------------
Requirements:
-----------------------
1. install netifaces using pip
command: pip install netifaces
